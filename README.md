To download your WhatsApp group chat history, use WhatsApp’s built-in Export Chat feature on your phone. This works the same way for both individual and group conversations.

Before exporting the history chat, make sure that the advanced privacy is not turned on. If so turn it off from the info group section.

Steps to Export on Android
1.	Open WhatsApp and navigate to the group chat you want to download
2.	Tap the three vertical dots (More options) in the top right corner
3.	Select More > Export chat
4.	Choose Without media (text only) or Include media (includes photos, videos, voice messages)[pinktape +1]
5.	Select where to save or send the export—options include email, Google Drive, your phone’s files, or other apps
